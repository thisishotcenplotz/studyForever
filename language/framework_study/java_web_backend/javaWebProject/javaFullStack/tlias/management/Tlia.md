### 日志级别

| **日志级别** | **说明** |
|:--------:|:-----|
| trace    | 追踪，记录程序运行轨迹 |
| debug    | 调试，记录程序调试过程中的信息，实际应用中一般将其视为最低级别 |
| info     | 记录一般信息，描述程序运行的关键事件，如：网络链接、IO操作等 |
| warn     | 警告信息，记录潜在有害的情况 |
| error    | 错误信息 |
